# Origin-Design

**AI 图片 & 视频创作工作流平台**

可视化节点编辑器，基于 Agnes AI 实现图像生成、图像编辑和视频创作。

</div>

---

## 功能

- **Text to Image** — 文本生成图像
- **Image to Image** — 以图生图/编辑
- **Text to Video** — 文本生成视频
- **Image to Video** — 图片生成视频
- **Start/End Frame Video** — 首尾帧视频生成
- **创意描述** — 扩展和优化提示词

## 快速开始

```bash
npm install
npm run dev
```

在设置中配置 Agnes AI API Key 即可使用。

## 技术栈

- React 19 + TypeScript
- Vite 6
- Tailwind CSS
- Agnes AI API (agnes-image, agnes-video)
- React Flow 节点编辑器
